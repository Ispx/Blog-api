# Blog Api

Uma api REST para criação de noticias desenvolvida em dart.


## Rodar a aplicação

#### Clone o repositório

$ git clone https://github.com/Ispx/Blog-api.git

#### Navegue até o arquivo do projeto

$ cd blog-api

#### Criar e iniciar contêineres

$ docker-compose up --build


## Tecnologias

* Shelf
* Shelf-router
* Password-dart
* Jaguar-jwt
* Mysql1
